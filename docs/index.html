Tools & Technologies

The project is developed in Xcode 10.1, iOS 12 SDK, Swift 4.2 and minimum deployment target as iOS 11.

OS & Devices: iOS 11+, iPhone and iPad, however the UX is not optimized for small screens like iPhone SE and iPad but the app is functional

Programming approach: ReactiveX using RxSwift and RxCocoa

Architecture: MVVM with only Root Coordinator

Dependency Manager: Cocoapods

Dependecies: RxSwift, Alamofire, NVActivityIndicatorView, RxMKMapView

Code documentation: Using jazzy

Core Modules: FSCoreKit, FSUIKit

Test using: XCTest and RxBlocking

Lint using: Swiftlint

Custom views:

Description

Below is an overview of the Functionality implemented:

  • Fetch and display the list of vehicles
  • User can see vehicle pins on the map
  • User can see tap on a pin to see details
  • Fetch the vehicle details on tap of the pin
  • Dismiss the vehicle details on tap outside on the map

FSCoreKit

This is a group of classes and extensions taken for commonly used functionality across diffent projects. This module is not specific to Flash Scooter App and can be used in other projects

FSUIKit

This is also a group of classes and extensions but specific to the UI functionality
